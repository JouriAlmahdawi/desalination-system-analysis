Seawater Desalination System Analysis

Project Description
This project analyzes the performance of a seawater desalination system using Python.
The analysis studies how physical variables affect water production and system efficiency.

Variables used
Salinity (ppm)
Osmotic pressure (bar)
Operating pressure (bar)
Energy consumption (kWh)
Water production (m³/day)

Goals
Understand relationships between physical variables in desalination systems
Analyze how salinity affects osmotic pressure
Study how operating pressure influences water production
Evaluate system efficiency and energy consumption
Visualize system behavior using data plots

Tools
Python
Pandas
Matplotlib

Author
Physics Student
